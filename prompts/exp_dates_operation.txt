You are tasked with determining the operations start date for each phase of a clean technology project based on a company’s investment into a facility. The clean tech project is in {location}, {country} and ran by {company}. Your colleague has identified that there are {value} investment phases for this project.

Please return a Python list of the operations start dates for each phase, corresponding to the chronological order of the phases, beginning with the initial phase and continuing with the expansion phases. If the exact date is not available but a time period is discussed, return a date from the middle of the period. 

If operations of a particular phase has not started, or it has started but the date is not available, please return "NA" for that item. The list should be {value} items long, in line with the number of phases which your colleague has identified.

The format for the date should be “MM-YYYY”. Please it is vitally important that you reply ONLY with a list of the dates. Return a list of strings with **class=list** and **size={value}**, where **{value}** represents the total number of phases. 

An example return is: ["03-2022", "07-2023"] or ["01-2023", "01-2023", "09-2022"]
