**crawl.py** is the main file to run the crawler. 
- max_pages, and page types to be crawled where appropriate are defined here

crawler functions are defined for each website in the **crawlers** folder.
Crawlers are named after the website they are scraping, currently defined for: 
- **Electrive**
- **Offshore Wind Biz**
- **Power Technology**
- **PV Magazine**
- **PV Tech**
- **Renews Biz**
- **World Energy**

all outputs are stored in the **URLs** folder. This is the freshest list of URLs available for the model.
